# GestionDeVentanasJavaFX

Proyecto desarrollado en Java utilizando JavaFX y Maven para la gestión de múltiples ventanas en aplicaciones de escritorio.

## Descripción

La aplicación está orientada a la gestión de ventanas en JavaFX, permitiendo abrir, cerrar y cambiar entre diferentes escenas o vistas dentro de la misma aplicación.

Este proyecto sirve como práctica para entender cómo se organizan y controlan múltiples ventanas en aplicaciones JavaFX.

## Tecnologías utilizadas

- Java
- JavaFX
- Maven

## Requisitos

- JDK 17 o superior
- Maven instalado
- IDE compatible con JavaFX (IntelliJ IDEA, Eclipse o NetBeans)
